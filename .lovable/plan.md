

# إضافة ملف vercel.json للنشر على Vercel

سيتم إنشاء ملف `vercel.json` في جذر المشروع لتسهيل النشر على Vercel مع إعدادات مناسبة لتطبيق React SPA:

### الملف: `vercel.json`
- إعادة توجيه جميع المسارات إلى `index.html` (ضروري لـ React Router)
- تحديد أمر البناء ومجلد الإخراج
- إعدادات الهيدرز للأمان والكاش

### ملاحظة
ملف `vite.config.ts` يحتوي بالفعل على `base: "/"` وهو مطلوب لـ Vercel.
